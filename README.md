# 亚健康管理系统
---
## 环境：
#### 基于SpringBoot3，MyBatis-plus，MySQL，Vue2，Element-UI等，实现前后端分离
#### 运行需配置Java，MySQL，Node， Vue等环境
- ##### Java：17
- ##### MySQL：8
- ##### Node：20
- ##### Vue：2
---
## 项目介绍：
#### 本项目旨在制作一个集合化的健康管理系统，将用户信息、运动知识、用户身体信息等管理系统集成，并实现了健康评分，实现强大的健康管理。
